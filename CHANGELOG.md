# Changelog / 版本说明

- 中文：更新并扩充雅思阅读题库，补充新的题目内容，持续优化题库数据与索引覆盖，方便更高效地检索与练习。
- English: Updated and expanded the IELTS reading question bank with new content, while continuing to improve data quality and index coverage for faster search and practice.

- 中文：在练习场景中加入 AI 助教，支持提示、思路讲解、错题复盘等学习辅助能力，帮助用户更高效地理解题目与完成复习。
- English: Added an AI coach to practice sessions, with support for hints, reasoning walkthroughs, and mistake review to help users understand questions and revise more efficiently.
